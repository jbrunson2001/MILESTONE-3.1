About the Author

Joshua Brunson

I am a Kennesaw State University student studying Cybersecurity. This project was created for IT 3203: Introduction to Web Development.

I chose browsers and browser engines because nearly everyone uses a browser, but the engine behind the browser is easy to overlook. The topic connects directly to HTML, CSS, JavaScript, web standards, security, and responsive design.

Project goals

  • Explain the difference between a browser and browser engine
  • Present organized research with reliable sources
  • Use semantic HTML and one external CSS file
  • Use an external JavaScript file for the quiz
  • Make every page responsive and mobile friendly
  • Practice testing at desktop, tablet, and phone sizes

How the project developed

Milestone 1

Built the original multi-page research website using HTML and CSS.

Milestone 2

Added the JavaScript self-assessment quiz, score results, and reset option.

Milestone 3

Expanded the research, improved the layout, and added responsive rules for desktop, tablet, and mobile screens.
