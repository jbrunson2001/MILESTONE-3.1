Research Sources

References and Resources

These sources support the history, technical explanations, current market information, responsive design choices, and future direction discussed in this website. Each source includes a short comment explaining how it was used.

  1. CERN. A short history of the Web.

    Explains the first browser and web server created by Tim Berners-Lee at CERN.

  2. NCSA. NCSA Mosaic.

    Describes Mosaic and its role in making graphical web browsing easier and more popular.

  3. Mozilla. History of the Mozilla Project.

    Provides background on Netscape releasing its source code and the beginning of Mozilla.

  4. Mozilla. Gecko - Firefox Source Docs.

    Official technical overview of Mozilla's Gecko rendering engine.

  5. Chromium. Blink rendering engine.

    Official Chromium explanation of Blink and its place in the Chromium source code.

  6. Chromium Blog. Blink: A rendering engine for the Chromium project.

    Announcement explaining why Chromium created Blink from WebKit in 2013.

  7. WebKit. The WebKit Open Source Project.

    Explains WebKit's purpose, project goals, standards focus, and use as a web content engine.

  8. WebKit. Announcing Interop 2026.

    Describes current collaboration among engine teams to improve cross-browser interoperability.

  9. WHATWG. HTML Living Standard.

    The current technical standard for HTML and many browser behaviors.

  10. W3C. Web Standards.

    Explains the role of shared web standards in browsers and other web software.

  11. W3C Web Accessibility Initiative. WCAG overview.

    Introduces accessibility standards used to make web content more usable for people with disabilities.

  12. MDN Web Docs. Responsive web design.

    Explains responsive layouts, flexible media, and mobile-first design.

  13. MDN Web Docs. Media query fundamentals.

    Explains CSS media queries and breakpoint-based layout changes.

  14. StatCounter Global Stats. Browser market share worldwide.

    Provides current worldwide browser market share data used on the Home page.

  15. web.dev. Interop 2026: Continuing to improve the web for developers.

    Summarizes the 2026 cross-browser effort to improve feature compatibility.
